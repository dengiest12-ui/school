# School Class Family App

Мобильное приложение для жизни школьного класса: ДЗ, расписание, чат, объявления, события, сборы, фотоальбомы и напоминания без хаоса в родительских чатах.

## Цель продукта

Сделать повседневную школьную логистику понятной для родителей:

- что завтра в школу;
- что ребенку нужно сделать;
- что родителю нужно оплатить, подписать, купить или принести;
- что важного произошло в классе.

## MVP-фокус

- iPhone-приложение, iOS 17+.
- Русский интерфейс.
- Класс работает даже без активного учителя.
- Главный экран: "Сегодня / Что завтра".
- ДЗ вручную и по фото.
- Чат и объявления.
- Календарь и расписание.
- Сборы без приема денег.
- Напоминания.
- Семейная подписка.

## Рабочий принцип

Сначала фиксируем границы MVP и проверяем главный сценарий:

1. Родитель или родкомитет создает комнату класса.
2. Приглашает родителей.
3. Добавляет расписание.
4. Добавляет ДЗ вручную или по фото.
5. Создает событие или сбор.
6. Родители получают понятное "что завтра".
